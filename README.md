# raknalang (räknalang)

A simple educational programming language for learning basic arithmetics in native language.

## Operators

| symbol | operator     | swedish       	|
| :----: | --------     | --------      	|
| --     | COMMENT      | kommentar     	|
| +      | PLUS         | plus          	|
| -      | MINUS        | minus          	|
| >      | GREATER THAN | fler, mer, större |
| <      | LESS THAN 	| färre, mindre 	|
| /      | DIVIDE       | delat, delad      |
| *      | MULTIPLY     | gånger 	        |
